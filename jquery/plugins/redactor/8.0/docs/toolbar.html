Toolbar

By default, Redactor has the following set of buttons:

  1. ['html', '|', 'formatting', '|', 'bold', 'italic', 'deleted', '|',
  2. 'unorderedlist', 'orderedlist', 'outdent', 'indent', '|',
  3. 'image', 'video', 'file', 'table', 'link', '|',
  4. 'fontcolor', 'backcolor', '|', 
  5. 'alignleft', 'aligncenter', 'alignright', 'justify', '|',
  6. 'horizontalrule']

If you need to setup your own set of buttons you can do so using this option:

  1. $('#redactor').redactor({
  2.     buttons: ['formatting', '|', 'bold', 'italic'] 
  3. });

How to create a button

First, set an icon from any of the styles available to represent your new button, like this:

  1. <style type="text/css">
  2. body .redactor_toolbar li a.redactor_btn_button1  {
  3.     background: url(/img/button1.png) no-repeat;
  4. }
  5. </style>

The button size is 25x25px (and 50x50px for retina).

Now you can call Redactor with the new button specified in an array:

  1. $(document).ready(
  2.     function()
  3.     {
  4.         var buttons = ['formatting', '|', 'bold', 'italic', '|', 'button1'];
  5. 	
  6.         $('#redactor').redactor({
  7.             focus: true, 
  8.             buttons: buttons, 
  9.             buttonsCustom: {
  10.                 button1: {
  11.                     title: 'Button', 
  12.                     callback: function(obj, event, key) { … } 
  13.                 }
  14.             }
  15.         });
  16.     }
  17. );

Or you can add the new button in the array by default:

  1. $(document).ready(
  2.     function()
  3.     {		
  4.         $('#redactor').redactor({
  5.             focus: true, 
  6.             buttonsAdd: ['|', 'button1'], 
  7.             buttonsCustom: {
  8.                 button1: {
  9.                     title: 'Button', 
  10.                     callback: function(obj, event, key) { … }  
  11.                 }
  12.             }
  13.         });
  14.     }
  15. );

Obviously, the new button's function can be external:

  1. function testButton(obj, event, key)
  2. {
  3.     alert(key);
  4. }
  5.  
  6. $(document).ready(
  7.     function()
  8.     {		
  9.         $('#redactor').redactor({
  10.             focus: true, 
  11.             buttonsAdd: ['|', 'button1'], 
  12.             buttonsCustom: {
  13.                 button1: {
  14.                     title: 'Button', 
  15.                     callback: testButton 
  16.                 } 
  17.             }
  18.         });
  19.     }
  20. );
