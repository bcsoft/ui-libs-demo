API


getEditor

Returns an object of the Redactor's editable layer.

  1. $('#redactor').getEditor();

getCode

Returns the Redactor's HTML-code.

  1. $('#redactor').getCode();

setCode

Puts an HTML code into the Redactor.

  1. $('#redactor').setCode('<p>text</p>');

getText

Returns Redactor's html as plain text.

  1. $('#redactor').getText();

getSelected

Returns Redactor's selected html.

  1. $('#redactor').getSelected();

insertHtml

Puts an HTML code before the cursor.

  1. $('#redactor').insertHtml('<p>insert</p>');

destroyEditor

Recovers textarea by removing iframe.

  1. $('#redactor').destroyEditor();

setFocus

Sets a focus to the Redactor.

  1. $('#redactor').setFocus();

execCommand

This option allow you to execute the execCommand.

  1. $('#redactor').execCommand('bold');
  2. $('#redactor').execCommand('formatblock', '<h5>');
  3. $('#redactor').execCommand('inserthtml', '<p>text</p>');
